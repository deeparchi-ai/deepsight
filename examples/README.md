# Examples

Worked examples of DeepSight, so a reader can see the machinery actually run —
and so there is a concrete fixture to eyeball against when the methodology
changes (the repo otherwise has no way to verify the prompts behave as claimed).

| File | Mode | Shows |
|------|------|-------|
| [`lightweight-walkthrough.md`](lightweight-walkthrough.md) | 轻量 | Phase 0 路由 → 约束追问 → 对抗审查 → `[A/B/C]` 证据分级 → 决策刹车 |

> These walkthroughs are **illustrative** — chosen on small, externally-checkable
> questions so the evidence tiers are honestly assignable, not fabricated deep
> research. They demonstrate *the process*, not authoritative conclusions.
